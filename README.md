# NotebookLM Sources Hub

A curated directory of websites hosting free files and knowledge for [Google NotebookLM](https://notebooklm.google) sources.

🔗 **Live site:** https://pegatanc.github.io/notebooklm-sources-hub/

## What is this?

A single-page directory of websites where you can download free PDFs, ebooks, academic papers, datasets, and other files to use as sources in your NotebookLM notebooks.

## Categories

- 📄 **Academic Papers & Research** — arXiv, Semantic Scholar, Google Scholar, CORE, DOAJ, PubMed, bioRxiv, SSRN, Zenodo
- 📚 **Free Textbooks** — OpenStax, LibreTexts, OAPEN, Open Textbook Library, Bookboon, BCcampus, Saylor, IntechOpen, OER Commons
- 📖 **General Knowledge & Ebooks** — Internet Archive, Project Gutenberg, Wikibooks, PDF Drive, Manybooks, Open Culture, Digital Book Index
- 📊 **Datasets & Data** — Kaggle, Google Dataset Search, HuggingFace, Data.gov
- ⚙️ **NotebookLM Native** — Discover Sources, Deep Research, Google Drive, YouTube, Public Notebooks

## How to use

1. Find a site from the directory above
2. Download files (PDF, EPUB, text, audio, CSV)
3. Open [NotebookLM](https://notebooklm.google)
4. Click "Add source" → upload file or paste URL
5. Limits: 50 sources/notebook (300 for Plus), max 500KB text, 200MB per source

## Tech

Single `index.html`. Zero dependencies. Zero build step. Vanilla HTML/CSS/JS.

## Contributing
